pdnsd Download Page

The original author of pdnsd is Thomas Moestl, but since 2003, he no longer maintains pdnsd. However, Paul Rombouts has extensively revised the code and maintains a version with many fixes and improvements at http://www.phys.uu.nl/~rombouts/pdnsd.html.
He has pre-patched tarballs and RPM packages available for download at this site.
There is also a mirror site at http://members.home.nl/p.a.rombouts/pdnsd.html.

The most recent tarball is pdnsd-1.2.7-par.tar.gz (GPG signature).
The most recent RPM packages are:
Package Name Size Description
pdnsd-1.2.7-par.src.rpm 457kB Source RPM (binary packages can also be built directly from the tarball).
pdnsd-1.2.7-par_sl5.i386.rpm 325kB i386 binary built on a Scientific Linux 5.2 system (compatible with Red Hat EL 5).
pdnsd-1.2.7-par_sl5.x86_64.rpm 328kB x86_64 binary built on a Scientific Linux 5.2 system (compatible with Red Hat EL 5).
If you want to check the signatures on these packages you will need a copy of my GPG key which you can get here or from a public key server.

There are also Debian, Gentoo, Mandriva and FreeBSD packages, but these may not include the latest version.
If you are looking for other versions released by Paul Rombouts, visit this download directory.
